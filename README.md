# Countdown Timer

Implemented with Win32 API.

# Usage

| Enter | Start a countdown |
| Esc | Cancel the countdown / Quit |
| Shift | Toggle 30 seconds |
| Right | +1 minute |
| Left | -1 minute |
| Up | +10 minutes |
| Down | -10 minutes |
| Page-Up | +1 hour |
| Page-Down | +1 hour |
| Delete / Backspace | Set to 0 second |
| Mouse-Scroll | Change the size | 

# To build
Run the command "cabal build".
